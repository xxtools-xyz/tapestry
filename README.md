# 🌟 Tapestry Journal Downloader

A lightweight desktop application for parents to download, sync, and beautifully preserve their children's [Tapestry](https://tapestryjournal.com/) learning journal observations — including photos, videos, and teacher notes — as an offline, browsable HTML journal.

---

## ✨ Features

- **🔐 Secure In-App Login** — Log in directly through Tapestry's official page embedded inside the app. No need to share your Tapestry password.
- **👶 Multi-Child Support** — Automatically detects all children linked to your account. Each child gets a separate, isolated data folder.
- **🔄 Incremental Sync** — Only downloads new observations since your last sync. Resume-safe with `.part` file support for large videos.
- **📅 Timestamp Preservation** — Downloaded files have their creation/modification dates set to the original observation date, so they sort correctly in Google Photos, iCloud, etc.
- **📖 Offline HTML Journal** — Automatically generates a beautiful, month-by-month HTML journal you can open in any browser — complete with image zoom, video playback, and a warm cartoonish aesthetic.
- **📂 Smart File Organization** — Files are neatly organized by `Child Name / Year-Month / media/` with full metadata preserved.

## 🚀 Getting Started

1. Download the application from the provided download link.
2. Open the application and log in using your email to access your Tapestry Journal Downloader account.
3. Enter your school's Tapestry prefix (e.g. `nis-jakarta`) or full URL and log into Tapestry.
4. Select your child and choose a download destination folder.
5. Hit **Start Full Sync** to begin archiving your memories!

## 🛡️ Privacy & Security

- Your Tapestry credentials are only ever entered on Tapestry's official login page.
- The app extracts a session cookie from the embedded browser — we never see or store your Tapestry password.
- All downloaded journals and media are saved locally on your machine.
- User accounts and subscription data are securely managed through industry-standard authentication and payment providers.
